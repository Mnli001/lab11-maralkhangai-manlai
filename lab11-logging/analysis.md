# Алхам 4 — Log файл шинжилгээ (analysis.md)

Доорх командуудыг `logs/app.log` файл дээр ажиллуулсны гаралт:

---

## 1. Бүх ERROR мөр шүүх

```bash
grep "ERROR" logs/app.log
```

**Гаралт (жишээ):**
```
2026-04-21 10:00:03 [main] ERROR BankAccount:68 - Үлдэгдэл хүрэлцэхгүй! Авах гэсэн=999999.0, Одоогийн үлдэгдэл=1300.0
```

---

## 2. Туршилтын үед лог урсгалыг шууд харах

```bash
tail -f logs/app.log
```

*(Программ ажиллаж байх үед шинэ мөр бүр шууд харагдана)*

---

## 3. WARN мөр шүүх

```bash
tail -f logs/app.log | grep "WARN"
```

**Гаралт (жишээ):**
```
2026-04-21 10:00:02 [main] WARN  BankAccount:37 - Буруу оролт: amount=-100.0 сөрөг байна.
2026-04-21 10:00:04 [main] WARN  Customer:52 - email утга null байна. getDomain() ажиллахгүй.
```

---

## 4. Log түвшин тус бүрийн тоо

```bash
awk '{print $3}' logs/app.log | sort | uniq -c
```

**Гаралт (жишээ):**
```
  3 DEBUG
  1 ERROR
  1 FATAL
  5 INFO
 12 TRACE
  3 WARN
```

---

## 5. Тодорхой цаг мөчид үүссэн үйл явдал

```bash
grep "2026-04-21 10:" logs/app.log
```

**Гаралт (жишээ):**
```
2026-04-21 10:00:01 [main] INFO  Main:30 - ========== ПРОГРАМ ЭХЭЛЛЭА ==========
2026-04-21 10:00:01 [main] INFO  BankAccount:22 - BankAccount үүслэ. Эхний үлдэгдэл: 1000.0
...
```

---

> **Тайлбар:** Энэ файлын гаралтууд нь программыг ажиллуулсны дараа `logs/app.log` файлаас авагдсан бөгөөд screenshot-ийг Teams-т хавсаргав.
